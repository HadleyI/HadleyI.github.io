Chat Server

A C# program built in Advanced Object Oriented Programming. This Chat server was built up over the span of the course. The application used TCIP, netcat (as the server), a logging interface to track all chats, and multi-threading.

Planet Quiz

An Andriod applcation built in mobile application development class. This application consisted of ten questions read in from a CSV file. The questions and answers are shown at random, meaning the quiz will be different everytime it is run.

Media Player

This media player was written in C# and XAML using Windows Presentation Foundation (WPF). The application allows a user to select an MP3 file via the file explorer. Once a song is select, the title, artist, album, album art and year of release will be displayed. The user has the ability to start, stop or pause the current song. The user is also able to update the tags of the current song and save it back to the file.

Black Jack

This game of black jack is built in using javascript in client side programming class. The cards are displayed from an API call from deck Of Cards API. This very simple game of black jack allows the dealer or the player to hit or stay at anytime until one of them either gets 21 or busts.
